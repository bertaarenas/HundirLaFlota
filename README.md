# HundirLaFlota
Este pràctica està basada en el juego de Hundir la Flota, esta hecho para que jueguen dos personas y el tablero tiene un tamaño de 7x7.
Al empezar cada jugador coloca 4 barcos que ocupan 1 casilla, y en cada turno el jugador introduce una la fila y columna del tablero rival.
El juego acaba cuando uno de los dos jugadores se quede sin barco.
